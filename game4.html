About THE Game

Minesweeper game is played by one player, it is about a gum field and the player has to move in this field (pressing on the blocks), if the block shows a number, the number is about how many bombs beside this block, if the block shows a bomb then player lose.Black blocks are safety and not needed to be pressed.The player wins when he opens all blocks and place the bombs with flags by right click. The number of bombs is as the number of flagfs which is 7.
To Restart The game press on the emoji face that is beside the time on the right and the flag's number on the left.
